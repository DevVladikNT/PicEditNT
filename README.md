# PicEditNT

This is my android app for editing pictures.

It has some filters:
- Pixel
- Dark
- Circuit
- DarkYellow
- Neon
- BlackWhite
- Lining
- ColorShifts
- Defocusing
- Cyberpunk
- Petrol

Moreover, you can setup your own filter.

### Others

You need Android v8.0 and higher on mobile for correct work.<br>
